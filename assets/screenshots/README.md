## Dicas de captura

1. **Resolução:** desktop em 1440×900 ou 1920×1080; mobile em 390×844 (ou device frame).
2. **Navegador limpo:** ocultar favoritos; preferir modo anônimo se quiser UI do browser mínima.
3. **Estado “scrolled”:** uma captura do header com sombra após scroll fica bem no case.
4. **FAQ:** deixe **um** item aberto para mostrar o accordion.
5. **Formulário:** preencha com dados fictícios óbvios (`teste@exemplo.com`) e tire o estado de sucesso.
6. **Formato:** PNG para nitidez; WebP se quiser peso menor no site de portfólio.
7. **Nomeação:** use o prefixo numérico acima para ordem no Git e no README.

## Como citar no portfólio

Sugestão de legenda:

> Landing page premium (HTML/CSS/JS) para personagem fictícia de psicologia clínica.  
> Projeto demonstrativo — formulário sem envio real de dados.

## Opcional

- GIF curto (3–5s) do reveal no scroll ou do contador do hero  
- Vídeo Loom de 30–60s navegando a página  

Não é obrigatório versionar capturas pesadas no Git; você pode manter só este README e hospedar imagens no site de portfólio.
